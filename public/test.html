Login API Documentation

Endpoint

/login

Method

POST

Description

사용자 로그인 요청을 처리하는 API입니다.

Request Headers

  • Content-Type: application/json

Request Body

{
  "email": "user@example.com",
  "password": "secretpassword"
}

Response

200 OK

로그인 성공 시 반환되는 응답

{
  "email": "user@example.com",
  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9",
  "message": "로그인에 성공했습니다."
}

401 Unauthorized

해당하는 이메일이 없는 경우 반환되는 응답

{
  "message": "해당하는 이메일이 없습니다."
}

402 Forbidden

비밀번호가 일치하지 않는 경우 반환되는 응답

{
  "message": "비밀번호가 일치하지 않습니다."
}

Error Response

400 Bad Request

{
  "message": "잘못된 요청입니다."
}
